¿Qué es fepro?

FEPRO es una feria de proyectos y competencia universitaria donde los estudiantes desarrollan habilidades técnicas y de innovación.

Categorías

Aplicaciones Básicas

● Software Básico:
Aplicaciones de escritorio con GUI sencilla o de consola/terminal; debe utilizar estructuras de datos y/o manejo de archivos.

● Aplicaciones Web Básicas:
Aplicaciones ejecutables sobre cualquier navegador Web que implementen el modelo de comunicación cliente-servidor.

● Aplicacione Móviles Básicas:
Aplicaciones sencillas ejecutables sobre dispositivos móviles. Deben representar una propuesta novedosa.

Aplicaciones Avanzadas

● Software Especializado:
Que se hayan empleado conceptos de usabilidad e Ingeniería de Software para su desarrollo. Deben resolver algún problema complejo de la ciencia o la industria utilizando estructuras de datos abstractas, análisis de datos, IA, servicios en la nube, etc.

● Aplicaciones Web Especializadas:
Que implementen el modelo cliente-servidor. Deben ofrecer algún tipo de servicio en línea. Debe hacer uso de conceptos de Ingeniería Web.

● Aplicaciones Móviles Especializadas:
Aplicaciones robustas con conexión a internet y a bases de datos. Deben ofrecer alguna solución novedosa que justifique su utilidad móvil. Pueden consumir servicios Web y deben contar con una interfaz intuitiva y funcional.

● Internet de las Cosas (IOT):
Aplicaciones o dispositivos complejos que permitan la interconexión digital de objetos cotidianos con Internet; deben integrar sensores inteligentes conectados a Internet a través de redes fijas o inalámbricas, por lo que se requiere la implementación de un sistema que perciba su ambiente de ejecución.

Gráficos por Computadora

● Simulaciones 3D:
Implementación de un sistema con ambientes gráficos (2D o 3D), ejecutables sobre PC o dispositivos móviles.

● Realidad Virtual:
Implementación de un sistema con ambientes gráficos 3D; deben sumergir completamente a los usuarios dentro de un mundo virtual, y deben tener un propósito específico (social, comercial, recreativo).

● Realidad Aumentada:
Implementación de un sistema que realice una composición armónica de imágenes (obtenidas desde una cámara) y objetos 2D/3D en tiempo real. Deben tener un propósito específico (social, comercial, recreativo).

● Videojuegos:
Implementación de un sistema con ambientes gráficos (2D o 3D), ejecutables sobre PC, consolas o dispositivos móviles; deben atrapar a los usuarios en conflictos estructurados aplicando el concepto de jugabilidad.

Hardware y Robótica

Entidades físicas que sean multifuncionales y reprogramables que involucren la aplicación de conocimientos en electrónica, mecánica, programación y/o robótica; deben solucionar algún problema complejo o realizar alguna tarea específica con cierto grado de dificultad.
